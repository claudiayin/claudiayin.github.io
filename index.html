What is GitHub?

GitHub is a collaboration platform that uses Git for versioning. GitHub is a popular place to share and contribute to open-source software



What is a Repository?

A repository is a project containing files and folders. A repository tracks versions of files and folders.


What is a Branch?

A branch is a parallel version of your repository. By default, your repository has one branch named main and it is considered to be the definitive branch. Creating additional branched allows you to copy the main branch of your repository and safely make changes without disrupting the main project. Many people use branches to work on specific features without affectig any other parts of the project
